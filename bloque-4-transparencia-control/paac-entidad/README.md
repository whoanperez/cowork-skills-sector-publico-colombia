# Plan Anticorrupción y de Atención al Ciudadano (PAAC)

Genera Plan Anticorrupción y de Atención al Ciudadano con 6 componentes clave: diagnóstico, políticas, programas de prevención, sistema de atención y mecanismos de evaluación. Cumple Ley 1474/2011 Art. 73.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos institucionales y diagnóstico de vulnerabilidades
3. Responde las preguntas guiadas sobre políticas y programas anticorrupción
4. Recibe tu archivo .xlsx con PAAC estructurado y validado

## Salida

- **Formato:** .xlsx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 1474/2011 (Estatuto Anticorrupción), Art. 73
- CONPES 3654 (Política de Servicio al Ciudadano)
- Decreto 1082/2015 (Gestión Administrativa)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
