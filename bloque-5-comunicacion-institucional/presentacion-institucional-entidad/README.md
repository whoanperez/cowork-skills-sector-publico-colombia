# Presentaciones Institucionales Accesibles

Genera presentaciones .pptx profesionales adaptadas a 5 audiencias diferentes: autoridades, ciudadanía, servidores públicos, cooperantes internacionales y medios de comunicación. Cumple WCAG para accesibilidad digital.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos institucionales y tema de presentación
3. Responde las preguntas guiadas sobre audiencia, objetivo y contenido clave
4. Recibe tu presentación .pptx accesible lista para uso y presentación

## Salida

- **Formato:** .pptx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- WCAG (Web Content Accessibility Guidelines)
- Ley 1519/2013 (Accesibilidad Digital)
- Decreto 1082/2015 (Gestión Administrativa)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
