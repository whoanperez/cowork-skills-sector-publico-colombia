# Conceptos Técnicos para Actos Administrativos

Genera conceptos técnicos que respaldan decisiones administrativas en 4 tipologías: viabilidad técnica, análisis de capacidad, evaluación de propuestas y recomendación de acción. Cumple GTC-185 y estándares de justificación administrativa.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona contexto de la decisión o sube documentación de soporte
3. Responde las preguntas guiadas sobre tipología, análisis y recomendación
4. Recibe tu documento .docx con concepto técnico fundamentado

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- GTC-185 (Guía de Trabajo Colombiana)
- Decreto 1082/2015 (Gestión Administrativa)
- Ley 489/1998 (Estatuto de la Administración Pública)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
