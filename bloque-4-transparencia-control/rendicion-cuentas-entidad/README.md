# Rendición de Cuentas Institucional

Genera reportes de rendición de cuentas con estructura CONPES 3654 que documentan cumplimiento de planes, logros de gestión, financieros y temas de transparencia. Facilita comunicación clara con ciudadanía sobre desempeño institucional.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona período, logros alcanzados e indicadores de desempeño
3. Responde las preguntas guiadas sobre gestión, resultados y retos
4. Recibe tu documento .docx listo para socialización pública

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- CONPES 3654 (Política de Servicio al Ciudadano)
- Ley 1474/2011 (Transparencia Administrativa)
- Decreto 1082/2015 (Gestión Administrativa)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
