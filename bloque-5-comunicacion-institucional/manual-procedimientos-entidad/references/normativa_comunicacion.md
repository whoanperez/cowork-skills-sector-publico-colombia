# Normativa de Referencia — Comunicación Institucional y Documentación Operativa

## 1. Presentaciones Institucionales

### Marco MIPG para presentaciones
Las presentaciones institucionales deben articularse con las 7 dimensiones del MIPG
(Decreto 1499/2017) cuando corresponda a reportes de gestión, comités o espacios
de rendición de cuentas.

### Estructura recomendada por tipo de audiencia

| Audiencia | Enfoque | Duración sugerida | Elementos clave |
|---|---|---|---|
| Directivos/Comité | Ejecutivo | 10-15 slides | KPIs, semáforos, decisiones pendientes |
| Equipo de trabajo | Operativo | 15-25 slides | Tareas, cronogramas, responsables |
| Ciudadanía | Informativo/accesible | 10-15 slides | Lenguaje claro, resultados tangibles, visualizaciones |
| Entes de control | Técnico-normativo | 15-20 slides | Evidencias, cumplimiento, indicadores |
| Aliados/cooperación | Estratégico | 10-12 slides | Valor público, impacto, articulación |

### Lineamientos de accesibilidad (Resolución 1519/2020 MinTIC)
- Contraste mínimo 4.5:1 entre texto y fondo
- Tamaño de fuente mínimo legible (24pt para proyección, 18pt para texto secundario)
- Texto alternativo en imágenes cuando se publique versión digital
- Evitar transmitir información solo por color (usar también formas o texto)

## 2. Manuales de Procedimientos

### Decreto 1083 de 2015 — Sector Función Pública
Las entidades deben documentar sus procesos y procedimientos conforme al
modelo de operación por procesos (MIPG Dimensión 3).

### Estructura de un manual de procedimientos (DAFP)

| Sección | Contenido |
|---|---|
| 1. Objetivo | Propósito del manual |
| 2. Alcance | Procesos y áreas cubiertas |
| 3. Definiciones | Glosario de términos técnicos |
| 4. Marco normativo | Normas que fundamentan los procedimientos |
| 5. Generalidades | Políticas y lineamientos transversales |
| 6. Procedimientos | Para cada procedimiento: objetivo, alcance, responsable, descripción de actividades, flujograma, documentos de referencia, registros |
| 7. Formatos y registros | Listado de formatos asociados con su código |
| 8. Control de cambios | Historial de versiones |

### Codificación de procesos
Cada entidad tiene su propio esquema de codificación:
- Tipo de proceso: Estratégico (E), Misional (M), Apoyo (A), Evaluación (V)
- Código del proceso: según mapa de procesos institucional
- Tipo de documento: Manual (MN), Procedimiento (PR), Guía (GU), Formato (FT), Instructivo (IN)
- Versión: 01, 02, 03...

Ejemplo: `GE-PR01` = Proceso de Gestión Estratégica, Procedimiento 01

### NTC-ISO 9001:2015 — Sistema de Gestión de Calidad
Las entidades con certificación ISO deben:
- Documentar procesos con entradas, salidas, responsables y recursos
- Incluir criterios de seguimiento, medición y evaluación
- Mantener información documentada con control de versiones

### NTCGP 1000:2009 — Norma Técnica de Calidad en la Gestión Pública
Establece requisitos de documentación para entidades públicas colombianas:
- Política y objetivos de calidad documentados
- Manual de calidad (o equivalente)
- Procedimientos documentados requeridos por la norma
- Registros requeridos por la norma

## 3. Cartas de Compromiso / Memorandos de Entendimiento

### Naturaleza jurídica
- **No son contratos** — no generan obligaciones patrimoniales
- Expresan voluntad de cooperación entre entidades o con terceros
- Pueden ser: Cartas de Intención, Memorandos de Entendimiento (MOU), Acuerdos Marco de Cooperación, Declaraciones Conjuntas

### Elementos de un memorando de entendimiento
1. Identificación de las partes (entidades, representantes legales)
2. Antecedentes y consideraciones
3. Objeto del memorando
4. Compromisos de cada parte (acciones específicas, NO obligaciones contractuales)
5. Plazo de vigencia
6. Mecanismos de seguimiento
7. Propiedad intelectual (si aplica)
8. Solución de controversias
9. Cláusula de no creación de obligaciones financieras
10. Firmas autorizadas

### Normativa aplicable
- **Ley 489/1998 Art. 95-96** — Convenios interadministrativos
- **Decreto 1082/2015** — Cuando el MOU derive en contrato posterior
- **GTC-185** — Formato y estructura de documentos oficiales
- **Ley 594/2000** — Gestión documental

## 4. Formato GTC-185 para documentos de comunicación

- Tamaño carta (21.59 × 27.94 cm)
- Márgenes: 3cm superior/izquierdo, 2.5cm inferior/derecho
- Fuente: Arial o Times New Roman 11-12pt cuerpo, 10pt tablas
- Tabla de contenido para documentos >5 páginas
- Numeración desde página 2

## 5. Gobierno Digital y Comunicación (Decreto 767/2022)

### Política de Gobierno Digital
Las comunicaciones institucionales digitales deben:
- Ser accesibles (WCAG 2.1 nivel AA)
- Estar disponibles en formatos abiertos cuando sea posible
- Cumplir con la Política de Seguridad Digital
- Respetar la identidad institucional de la entidad
