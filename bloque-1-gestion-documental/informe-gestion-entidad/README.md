# Informes de Gestión Periódicos

Genera informes de gestión que documentan avances en ejecución de planes, logros, indicadores y recursos utilizados. Alinea con MIPG y estándares de reporte institucional colombianos.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Sube datos de tu plan de acción o proporciona período y dependencia
3. Responde las preguntas guiadas sobre logros, indicadores y desafíos
4. Recibe tu documento .docx listo para presentación y archivo

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- MIPG (Marco Integrado de Planeación y Gestión)
- Ley 594/2000 (Gestión Documental)
- Decreto 1082/2015 (Gestión Administrativa)
- GTC-185 (Guía de Trabajo Colombiana)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
