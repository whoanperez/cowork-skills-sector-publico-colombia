# Oficios, Memorandos y Circulares

Genera documentos formales de comunicación interna y externa según estándares de gestión documental colombianos. Aplica la Ley 594/2000 y la GTC-185 para garantizar conformidad normativa y trazabilidad.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Sube tu plantilla de oficio institucional o proporciona datos del remitente
3. Responde las preguntas guiadas sobre destinatario, asunto y contenido
4. Recibe tu documento .docx listo para revisión y firma

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 594/2000 de Gestión Documental
- GTC-185 (Guía de Trabajo Colombiana)
- Decreto 1082/2015 (Gestión Administrativa)
- Ley 489/1998 (Estatuto de la Administración Pública)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
