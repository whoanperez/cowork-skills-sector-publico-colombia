# Informe de Control Interno

Genera informes de control interno que documentan el sistema de control implementado, evaluación de riesgos, hallazgos y recomendaciones en 4 modalidades. Cumple Ley 87/1993 y estándares COSO de control interno.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona información de auditorías realizadas y hallazgos identificados
3. Responde las preguntas guiadas sobre evaluación, impacto y recomendaciones
4. Recibe tu documento .docx con informe de control interno fundamentado

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 87/1993 (Normas de Control Interno)
- Decreto 1082/2015 (Gestión Administrativa)
- Marco COSO (Control Interno - Componentes)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
