# Diagnóstico MPIA-Bogotá y Cálculo de Índice de Conformidad Medida (ICM)

Genera diagnóstico integral de conformidad MPIA (Marco para la Modernización de la Gestión Pública) aplicable a Bogotá. Calcula Índice de Conformidad Medida (ICM) y emite dictamen de cumplimiento según Acuerdo 003/2025 CDTD.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos institucionales y procesos a diagnosticar
3. Responde las preguntas guiadas sobre cumplimiento normativo y capacidades
4. Recibe tu documento .docx con diagnóstico, ICM calculado y dictamen

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** N/A
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Acuerdo 003/2025 CDTD (Conformidad MPIA Bogotá)
- Decreto 1082/2015 (Gestión Pública)
- MIPG (Marco Integrado de Planeación y Gestión)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
