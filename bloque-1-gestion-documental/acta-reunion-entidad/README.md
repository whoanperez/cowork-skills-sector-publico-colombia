# Actas de Reunión con Compromisos

Genera actas de reunión formales que documentan asistentes, temas, decisiones y compromisos con responsables y fechas de seguimiento. Cumple GTC-185 y estándares de gestión documental colombiana.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos de la reunión (fecha, lugar, asistentes, orden del día)
3. Responde las preguntas guiadas sobre temas tratados y compromisos
4. Recibe tu documento .docx listo para firma y distribución

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- GTC-185 (Guía de Trabajo Colombiana)
- Ley 594/2000 (Gestión Documental)
- Decreto 1082/2015 (Gestión Administrativa)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
