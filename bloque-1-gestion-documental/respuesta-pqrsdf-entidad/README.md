# Respuestas PQRSDF con Control de Términos

Genera respuestas a Peticiones, Quejas, Reclamos, Solicitudes de Acceso a Información Pública, Denuncias y Felicitaciones con control automático de plazos y conformidad con la Ley 1755/2015. Asegura cumplimiento de términos legales y trazabilidad.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Sube el PQRSDF original o proporciona datos del registro de entrada
3. Responde las preguntas guiadas sobre tipo de solicitud, plazo y respuesta propuesta
4. Recibe tu documento .docx con validación de términos y firmas

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 1755/2015 (Derecho de Petición)
- Decreto 1082/2015 (Regulación de PQRSDF)
- Ley 1581/2012 (Protección de Datos Personales)
- Acuerdo 003/2025 CDTD (MPIA Bogotá)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
