# Informes de Supervisión Contractual

Genera informes de supervisión que documentan cumplimiento contractual, avance física y financiera, hallazgos y recomendaciones. Cumple Ley 1474/2011 de transparencia y estándares de supervisión en contratación pública.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos del contrato y período a supervisar
3. Responde las preguntas guiadas sobre cumplimiento, avance y hallazgos
4. Recibe tu documento .docx con informe listo para archivo y seguimiento

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 1474/2011 (Transparencia en Contratación)
- Decreto 1082/2015 (Contratación Pública)
- Ley 80/1993 (Estatuto General de Contratación)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
