# Plan de Acción Institucional

Genera planes de acción institucionales de 6 hojas con 21 columnas incluyendo objetivos, actividades, responsables, plazos, presupuesto e indicadores. Cumple con Decreto 612/2018 y MIPG.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos institucionales y objetivos estratégicos a alcanzar
3. Responde las preguntas guiadas sobre actividades, responsables y recursos
4. Recibe tu archivo .xlsx con todas las columnas y validaciones

## Salida

- **Formato:** .xlsx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Decreto 612/2018 (Disposiciones de PDD)
- MIPG (Marco Integrado de Planeación y Gestión)
- Ley 594/2000 (Gestión Documental)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
