# Manual de Procedimientos Institucionales

Genera manuales de procedimientos detallados según estándares DAFP que documentan procesos, responsables, flujos, tiempos y puntos de control. Alineado con ISO 9001 para calidad de gestión administrativa.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona listado de procedimientos o sube diagramas de flujo existentes
3. Responde las preguntas guiadas sobre procesos, responsables y tiempos
4. Recibe tu documento .docx con manual estructurado y normalizado

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- DAFP (Guía de Procedimientos Administrativos)
- ISO 9001 (Sistemas de Gestión de Calidad)
- Ley 594/2000 (Gestión Documental)
- Decreto 1082/2015 (Gestión Administrativa)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
