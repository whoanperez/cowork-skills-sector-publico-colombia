# Cartas de Compromiso y MOUs (Memorandos de Entendimiento)

Genera cartas de compromiso y Memorandos de Entendimiento (MOUs) en 4 tipologías: cooperación interinstitucional, participación ciudadana, compromisos técnicos y acuerdos de voluntades. Cumple Ley 489/1998 y estándares de acuerdos administrativos.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona datos de las partes y contexto del acuerdo
3. Responde las preguntas guiadas sobre tipología, términos y alcance
4. Recibe tu documento .docx listo para firma bilateral o multilateral

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Ley 489/1998 (Estatuto de la Administración Pública)
- Decreto 1082/2015 (Acuerdos Administrativos)
- Ley 594/2000 (Gestión Documental)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
