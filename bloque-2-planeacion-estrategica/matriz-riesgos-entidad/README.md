# Matriz de Riesgos 5x5 DAFP

Genera matrices de riesgos institucionales de 4 hojas con escala 5x5 según estándares DAFP. Incluye identificación, análisis cualitativo, valoración y planes de tratamiento conforme NTC-ISO 31000.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona riesgos identificados o sube análisis previo de riesgos
3. Responde las preguntas guiadas sobre probabilidad, impacto y controles
4. Recibe tu archivo .xlsx con matriz completa y planes de mitigación

## Salida

- **Formato:** .xlsx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- NTC-ISO 31000 (Gestión del Riesgo)
- DAFP (Guía de Análisis de Riesgos)
- Decreto 1082/2015 (Gestión Administrativa)
- Ley 87/1993 (Normas de Control Interno)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
