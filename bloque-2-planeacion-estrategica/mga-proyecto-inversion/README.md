# Fichas de Metodología General Ajustada (MGA) para Proyectos de Inversión

Genera fichas MGA según estándares DNP con 4 módulos y 65 preguntas clave. Documenta problema, justificación, alternativas, evaluación financiera y diseño de ejecución para proyectos de inversión pública.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Proporciona descripción del proyecto de inversión propuesto
3. Responde las preguntas guiadas sobre problema, alternativas y viabilidad
4. Recibe tu documento .docx con ficha MGA completa validada por DNP

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Medio
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- Guía Metodología General Ajustada (MGA) - DNP
- Decreto 1082/2015 (Gestión de Proyectos de Inversión)
- Ley 1473/2011 (Estatuto de Inversión Pública)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
