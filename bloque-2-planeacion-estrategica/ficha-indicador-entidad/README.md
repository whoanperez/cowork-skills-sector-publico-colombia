# Fichas de Indicadores SMART

Genera fichas de indicadores SMART estructuradas según estándares DAFP. Permite gestionar lotes de hasta 30 indicadores con fórmulas, metas, responsables y periodicidad de seguimiento.

## Uso rápido

1. Abre Cowork e instala esta skill
2. Sube listado de indicadores o proporciona objetivos estratégicos
3. Responde las preguntas guiadas sobre definición SMART y responsables
4. Recibe tu documento .docx con fichas completas y validadas

## Salida

- **Formato:** .docx
- **Riesgo MPIA:** Bajo
- **Conformidad:** Acuerdo 003/2025 CDTD ✓

## Normativa aplicable

- DAFP (Departamento Administrativo de la Función Pública) - Estándares de indicadores
- MIPG (Marco Integrado de Planeación y Gestión)
- Decreto 612/2018 (Disposiciones de PDD)

## Archivos

- `SKILL.md` — Instrucciones de la skill para Claude
- `references/` — Normativa de soporte

## Instalación

Comprime esta carpeta como `.skill` (zip renombrado) y arrástrala a Cowork, o copia el contenido de `SKILL.md` en Configuración → Skills → Crear nueva skill.
